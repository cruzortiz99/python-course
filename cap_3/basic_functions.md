## Imprimir en consola

```python
print("Hola mundo!")
```

## Solicitar datos desde consola

```python
user_name = input("Ingresar nombre: ")
```

## Sumar

```python
suma = sum([1, 2, 3])

print(suma)
```

## Máximo valor de lista

```python
maximo = max([1, 2, 3])

print(maximo)
```

## Mínimo valor de una lista

```python
minimo = min([1, 2, 3])

print(minimo)
```

## Ordenar

```python
sort = sorted([10,5,7,9])

inverso = sorted([1, 2, 3], reverse=True)

print(sort, inverso)
```

## Funciones constructoras

```python
# Lista transformada
map(lambda x: x, [1,2,3])
# Lista filtrada
filter(lambda x: true, [1,2,3])
# Tupla
tuple([1,2,3])
# Set o conjunto de elementos únicos
set([1,2,3])
# Lista invertida
reversed([1,2,3])
# Texto
str(47589658)
```
