## Asignación

```python
# Asignación simple
a = 15
print(a)

# output: a
# ---
# Asignación de un incremento
a = 15
a += 16
print(a)

# output: 31
# ---
# Asignación de un decremento
a = 15
a -= 8

# output: 7
# ---
# Asignación de un producto
a = 15
a *= 2
print(a)

# output: 30
# ---
# Asignación de una división
a = 20
a /= 2
print(a)

# output: 10
# ---
# Asignación de un modulo
a = 10
a %= 2
print(a)

# output: 0
# ---
# Asignación de una potencia
a = 10
a **= 2
print(a)

# output: 100
```

## Comparación

```python
# Igualdad
print(15 == "texto")

# output: False
# ---
# Desigualdad
print(15 != 15)

# output: False
# ---
# Mayor que
print(15 > 41)

# output: True
# ---
# Menor que
print(15 < 41)

# output: False
# ---
# Menor igual
print(15 <= 41)

# output: True
# ---
# Mayor igual que
print(15 >= 41)

# output: False
```
