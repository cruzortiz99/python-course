## If

```python
a = 15
b = 20

if (a > b):
  print(f"{a} es mayor que {b}")

```

## If-else

```python
a = 80
b = 30
if (a < b):
  print(f"{a} es menor que {b}")
else:
  print(f"{a} no es menor que {b}")
```

## if-elif-else

```python
a = 80
b = 80
if (a > b):
  print(f"{a} es mayor que {b}")
elif (a < b):
  print(f"{a} es menor que {b}")
else:
  print(f"{a} es igual que {b}")
```
